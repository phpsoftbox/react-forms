# @phpsoftbox/forms
Компонент для отрисовки форм из компонентов: 
- https://github.com/phpsoftbox/forms
- https://github.com/phpsoftbox/db-config
- https://github.com/phpsoftbox/requisites

React-утилиты для рендера schema-driven форм поверх `@phpsoftbox/react-softbox`.

## Экспорт

- `DynamicFormFields`
- `buildDynamicFormData`
- `isDynamicFieldRequired`
- `isDynamicValueFilled`
- вспомогательные типы/хелперы
